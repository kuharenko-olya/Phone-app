2. Алгоритм работы программы 

   Как должна работать программа?


1. При нажатии сбоку на кнопку вкл дисплей телефона.
   При повторном нажатии выкл.

2. При нажатии на круглую кнопку появляется меню с приложениями
   (у нас пока будет 9 приложений)

3. Первое приложение показывает alert c текстом like.

4. Второе приложение показывает input.
   
   Когда пользователь вводит значение ему возвращается результат посчитанный в квадрате.

5. Девятое приложение меняет фон телефона.
